Спотер

Спотер - прилад, що дозволяє виконувати контактне зварювання. Цей пристрій використовується для з'єднання тонких деталей, і виконання робіт по рихтуванню. Найчастіше спотер застосовується при відновлювальних роботах кузовної частини авто.

Ви можете купити спотер "Скорпіон", зателефонувавши за номером, для отримання якого натисніть кнопку "Отримати номер продавця"

Особливість "Скорпіону" - він працює при 220V
